# Глоссарий по биологии — Лечебное дело / Педиатрия

> Все термины, определения и понятия из конспекта и билетов 1–126 по методичке П.И. Шаповалова и лекциям. **★** — определение из официального глоссария методички. Всего: **455** терминов.

## Раздел I. Цитология и молекулярная биология (109)

| Термин | Определение |
| --- | --- |
| **Автотрофы** | Организмы, синтезирующие органику из неорганики (продуценты): фототрофы (энергия света) и хемотрофы (энергия окисления). |
| **Акросома** | Органоид головки сперматозоида с ферментами для проникновения через оболочки яйцеклетки. |
| **Активный транспорт** | Перенос веществ через мембрану против градиента с затратой энергии (АТФ): ионные насосы, напр. Na⁺/K⁺-насос. |
| **Амитоз** | Прямое деление клетки перетяжкой без спирализации хромосом и образования веретена (часто в специализированных и повреждённых/опухолевых клетках). |
| **Антикодон** | Триплет нуклеотидов тРНК, комплементарный кодону иРНК. |
| **Аппарат Гольджи** | Органоид упаковки, модификации и сортировки белков, образования лизосом; структурная единица — диктиосома. |
| ★ **Ассимиляция (анаболизм)** | Совокупность процессов синтеза сложных органических веществ из простых, идущих с затратой энергии. |
| **АТФ** | Аденозинтрифосфат — мононуклеотид (аденин + рибоза + 3 остатка фосфорной кислоты); универсальный аккумулятор энергии клетки. |
| **Аутосомы** | Неполовые хромосомы (у человека 22 пары). |
| **Биваленты (тетрады)** | Пары конъюгировавших гомологичных хромосом, состоящие из 4 хроматид. |
| **Биология** | Наука о жизни как особом явлении природы; представляет собой систему (комплекс) биологических дисциплин. Термин ввели независимо Ламарк и Тревиранус (нач. XIX в.). |
| **Биохимическая эволюция** | Ведущая гипотеза происхождения жизни (Опарин–Холдейн): абиогенный синтез органики в «первичном бульоне» → коацерваты → первые клетки. |
| **Брожение** | Анаэробное превращение пирувата (в молочную кислоту/спирт) без образования новых молекул АТФ. |
| **Включения** | Непостоянные компоненты цитоплазмы: трофические (жиры, углеводы, белки), секреторные, экскреторные. |
| **Вторичные половые признаки** | Признаки, не участвующие в размножении, проявляющиеся в пубертате (оволосение, тембр голоса, молочные железы). |
| ★ **Гаметогенез (прогенез)** | Процесс образования и созревания гамет, осуществляющийся в половых железах. |
| ★ **Ген** | Участок молекулы ДНК (или РНК), кодирующий последовательность аминокислот в полипептидной цепи либо последовательность нуклеотидов в тРНК и рРНК. |
| **Ген-регулятор** | Ген, расположенный вне оперона, кодирующий белок-репрессор. |
| **Генетический код** | Система записи наследственной информации триплетами нуклеотидов. Свойства: триплетный, универсальный, вырожденный, неперекрывающийся, непрерывный, линейный, коллинеарный. |
| **Геном** | Вся совокупность наследственной информации организма (гаплоидный набор). У человека кодирующая часть — лишь ~1,1–1,4 %. |
| **Гетеротрофы** | Организмы, питающиеся готовыми органическими веществами (консументы, редуценты). |
| **Гетерохроматин** | Спирализованный, неактивный хроматин (на периферии ядра): конститутивный (постоянный) и факультативный (тельце Барра). |
| **Гистоны** | Основные белки хромосом (5 фракций: Н1, Н2А, Н2В, Н3, Н4); образуют нуклеосомы, обеспечивая упаковку ДНК. |
| **Гликокаликс** | Надмембранный слой гликопротеинов и гликолипидов у животной клетки. ⚠ У растений снаружи — целлюлозная клеточная стенка. |
| **Гликолиз** | Бескислородное расщепление глюкозы в цитоплазме до 2 молекул пирувата (+2 АТФ); первый этап энергообмена. |
| **Денверская классификация** | Систематизация хромосом по размеру и положению центромеры; 7 групп A–G (1960). |
| ★ **Диссимиляция (катаболизм)** | Совокупность процессов расщепления органических веществ до простых с высвобождением энергии (часть запасается в АТФ). |
| **ДНК** | Двойная антипараллельная правозакрученная спираль из нуклеотидов; цепи соединены водородными связями по комплементарности (А=Т, Г≡Ц). Хранит наследственную информацию (Уотсон и Крик, 1953). |
| **Жидкостно-мозаичная модель** | Модель строения мембраны: подвижный бислой липидов с «мозаикой» встроенных белков (периферических, полуинтегральных, интегральных). |
| ★ **Жизнь** | Способ существования белковых тел и нуклеиновых кислот, существенный момент которого — постоянный обмен веществ со средой; с прекращением обмена жизнь прекращается. |
| **Инвагинационная теория** | Происхождение мембранных органоидов эукариот путём впячивания (инвагинации) оболочки аэробного прокариота. |
| **Интерфаза** | Период подготовки клетки к делению: G1 (рост, 2n2c), S (удвоение ДНК, 2n4c), G2 (синтез белка, удвоение центриолей, 2n4c). |
| **Интрон** | Некодирующий («молчащий») участок гена, удаляемый при сплайсинге. |
| **иРНК (мРНК)** | Информационная (матричная) РНК — матрица для синтеза белка на рибосоме. |
| **Кариогамия** | Слияние ядер (пронуклеусов) гамет при оплодотворении. |
| **Кариотип** | Систематизированный набор хромосом соматической клетки (число, форма, размеры). У человека 46 = 22 пары аутосом + 1 пара половых (46,XX или 46,XY). |
| **Клеточная теория** | Клетка — элементарная структурно-функциональная единица живого; клетки сходны по строению; возникают делением материнской; формируют многоклеточный организм. Авторы — Шлейден и Шванн (1838–39), дополнил Вирхов («всякая клетка от клетки»). |
| **Клеточный (жизненный) цикл** | Период жизни клетки от её образования делением материнской до собственного деления или гибели. |
| **Клеточный центр** | Немембранный органоид из 2 центриолей; образует веретено деления. |
| **Коацерваты** | Капли концентрированных органических полимеров, отграниченные от среды; обладали примитивным обменом, росли и делились (этап происхождения жизни; Опарин, 1924). |
| **Кодон** | Триплет нуклеотидов иРНК, кодирующий одну аминокислоту. |
| ★ **Комплементарность** | Принцип пространственного соответствия азотистых оснований: А с Т (2 водородные связи), Г с Ц (3 связи). Лежит в основе репликации и транскрипции. |
| **Конъюгация (гомологов)** | Попарное сближение гомологичных хромосом в профазе I мейоза (зиготена) с образованием бивалентов. |
| **Конъюгация (простейших)** | Форма полового процесса — временный обмен генетическим материалом между особями (инфузории). |
| **Копуляция** | Форма полового процесса — полное слияние двух особей (простейшие). |
| **Кристы** | Складки внутренней мембраны митохондрий, увеличивающие поверхность для ферментов дыхательной цепи. |
| ★ **Кроссинговер** | Процесс обмена одинаковыми участками между гомологичными хромосомами (в пахитене профазы I); источник комбинативной изменчивости. |
| ★ **Кэпирование иРНК** | Присоединение к 5′-концу иРНК «колпачка» (кэпа), защищающего её от разрушения и обеспечивающего узнавание рибосомой. |
| **Лизосомы** | Одномембранные пузырьки с гидролитическими ферментами (активны в кислой среде); внутриклеточное переваривание. |
| ★ **Мейоз** | Особый вид деления клеток, в результате которого формируются гаплоидные гаметы; 2 деления — I редукционное (число хромосом ↓ вдвое), II эквационное (как митоз). |
| **Микротрубочки** | Элементы цитоскелета из белка тубулина: опора клетки, нити веретена деления. |
| **Микрофиламенты** | Тонкие нити цитоскелета из белка актина: обеспечивают движение и форму клетки. |
| **Митоз** | Основной способ деления соматических клеток (профаза → метафаза → анафаза → телофаза); даёт 2 генетически идентичные клетки (2n2c). ⚠ В анафазе число хромосом временно 4n. |
| **Митохондрии** | Двумембранные органоиды аэробного дыхания и синтеза АТФ; внутренняя мембрана образует кристы; имеют кольцевую ДНК и рибосомы. |
| **Нуклеотид** | Мономер нуклеиновой кислоты из 3 частей: остаток фосфорной кислоты + пентоза (дезоксирибоза/рибоза) + азотистое основание. |
| ★ **Обмен веществ (метаболизм)** | Совокупность химических и физических превращений в организме, обеспечивающих жизнедеятельность во взаимосвязи со средой. Включает пластический и энергетический обмен. |
| **Овогенез** | Образование яйцеклеток; 3 стадии (размножение, рост, созревание); из 1 клетки — 1 яйцеклетка + 3 полярных тельца. Стадия формирования отсутствует. |
| **Оператор** | Участок оперона, связывающий белок-репрессор и регулирующий транскрипцию структурных генов. |
| **Оперон** | Структурно-функциональная единица генома прокариот: промотор → оператор → структурные гены (только экзоны) → терминатор (Жакоб и Моно, 1961). |
| ★ **Оплодотворение** | Процесс слияния женской и мужской половых клеток с образованием зиготы; восстанавливает диплоидность и объединяет наследственный материал родителей. |
| **Парижская классификация** | Систематизация хромосом по рисунку дифференциальной окраски (бэндинг); система символов (напр. 47,XY,+21 — синдром Дауна), 1971. |
| **Партеногенез** | Развитие организма из неоплодотворённой яйцеклетки (пчёлы-трутни, тли, дафнии). Бывает естественный/искусственный, гаплоидный/диплоидный. |
| **Пассивный транспорт** | Перенос веществ через мембрану по градиенту концентрации без затрат энергии: диффузия, осмос, облегчённая диффузия. |
| **Пероксисомы** | Мелкие органоиды с ферментом каталазой; обезвреживают перекись водорода (H₂O₂). |
| **Плазмолемма (клеточная мембрана)** | Поверхностная мембрана клетки по жидкостно-мозаичной модели: бислой липидов (гидрофильные головки наружу, гидрофобные хвосты внутрь) + белки + углеводы. Функции: барьер, транспорт, рецепция. |
| **Пластиды** | Двумембранные органоиды растений: хлоропласты (фотосинтез, хлорофилл), лейкопласты (запас), хромопласты (пигменты, окраска). |
| ★ **Полиаденилирование** | Присоединение к 3′-концу иРНК последовательности из 20–200 адениловых нуклеотидов (поли-А-хвост), повышающей стабильность. |
| **Политения** | Кратное увеличение числа копий ДНК в хромосоме при сохранении диплоидного набора (гигантские хромосомы слюнных желёз насекомых). |
| ★ **Половой диморфизм** | Различия между полами по морфологическим, биохимическим, физиологическим признакам, обеспечивающие реализацию репродуктивной функции. |
| **Половые хромосомы (гоносомы)** | Хромосомы, определяющие пол: X и Y. |
| **Прокариоты** | Организмы без оформленного ядра и мембранных органоидов (бактерии, цианобактерии); рибосомы 70S. |
| **Промотор** | Стартовый участок ДНК, к которому присоединяется РНК-полимераза для начала транскрипции. |
| **Процессинг** | Созревание про-иРНК в ядре эукариот: кэпирование → сплайсинг → полиаденилирование. |
| ★ **Размножение** | Универсальное свойство живого — способность воспроизводить себе подобных, обеспечивая непрерывность жизни. Бесполое (митоз) и половое (мейоз). |
| **Репликация** | Самоудвоение ДНК полуконсервативным способом перед делением клетки. Ферменты: геликаза, топоизомераза, ДНК-полимераза (только 5′→3′), лигаза. |
| **Репликон** | Участок ДНК от одной точки начала репликации до другой. |
| **Репрессор** | Белок, связывающийся с оператором и выключающий оперон (репрессия). |
| **Рибосомы** | Немембранные органоиды из рРНК и белка; место синтеза белка; 70S у прокариот, 80S у эукариот; группы на иРНК — полисомы. |
| **РНК** | Одноцепочечная нуклеиновая кислота (рибоза + фосфат + основание, У вместо Т). Виды: иРНК, тРНК, рРНК. |
| **рРНК** | Рибосомная РНК — структурная основа рибосомы; катализирует образование пептидных связей. |
| ★ **Сайленсер** | Участок ДНК, выключающий транскрипцию (ослабитель), изменяя структуру хроматина. |
| **Свойства живого** | Обмен веществ и энергии, самовоспроизведение, наследственность, изменчивость, рост и развитие, раздражимость, саморегуляция, дискретность и целостность. |
| **Симбиотическая теория** | Происхождение митохондрий, пластид и ресничек эукариот от симбиотических бактерий, поселившихся в клетке-хозяине. |
| **Сперматогенез** | Образование сперматозоидов; 4 стадии (размножение, рост, созревание, формирование); из 1 клетки — 4 полноценных сперматозоида. |
| ★ **Сплайсинг иРНК** | Вырезание интронов из про-иРНК и сшивание экзонов в зрелую иРНК. Альтернативный сплайсинг даёт полиморфизм белков. |
| **Тельце Барра** | Половой хроматин — инактивированная сконденсированная Х-хромосома в соматических клетках женского пола (компенсация дозы генов, гипотеза Лайон). |
| **Терминатор** | Концевой участок (стоп-последовательность), на котором завершается транскрипция. |
| **Транскриптон** | Структурно-функциональная единица генома эукариот: энхансер/сайленсер → промотор → структурные гены (экзоны + интроны) → терминатор. |
| ★ **Транскрипция** | Процесс записи (считывания) информации с молекулы ДНК на молекулу РНК. Фазы: инициация, элонгация, терминация. |
| ★ **Трансляция** | Процесс синтеза полипептидной цепи на основе информации иРНК. Фазы: инициация (старт-кодон АУГ), элонгация, терминация (стоп-кодон). |
| **тРНК** | Транспортная РНК («клеверный лист»): переносит аминокислоту, несёт антикодон. |
| **Уровни организации живого** | Иерархия живых систем: молекулярно-генетический → клеточный → тканевый → органный → организменный → популяционно-видовой → биогеоценотический → биосферный. |
| ★ **Фотосинтез** | Образование органических соединений из неорганических при участии энергии солнечного света (6CO₂+6H₂O→C₆H₁₂O₆+6O₂); главный источник кислорода атмосферы. |
| **Фрагменты Оказаки** | Короткие фрагменты, которыми синтезируется отстающая цепь ДНК (т.к. полимераза работает только 5′→3′); сшиваются лигазой. |
| **Хиазмы** | Точки перекреста хромосом — видимые места произошедшего кроссинговера (диплотена). |
| **Хроматин** | Комплекс ДНК с белками (гистонами) в ядре — деконденсированная форма хромосом. |
| **Хромосома** | Структура из ДНК и белков (нуклеопротеид), несущая гены. Метафазная хромосома = 2 хроматиды + центромера; плечи p (короткое) и q (длинное). |
| **Центромера** | Первичная перетяжка хромосомы, соединяющая сестринские хроматиды и связывающая нити веретена деления. |
| **Цитокинез (цитотомия)** | Разделение цитоплазмы при делении клетки: перетяжкой у животных, перегородкой у растений. |
| **Цитоплазма** | Внутренняя среда клетки (гиалоплазма с органоидами и включениями); способна к обратимым переходам золь ⇄ гель. |
| **Экзон** | Кодирующий (информативный) участок гена, сохраняющийся в зрелой иРНК. |
| **Эндомитоз** | Кратное увеличение числа хромосом без деления клетки (ведёт к полиплоидии). |
| **Эндоцитоз** | Захват веществ клеткой впячиванием мембраны: фагоцитоз (твёрдые частицы) и пиноцитоз (жидкость). |
| ★ **Энхансер** | Участок ДНК — усилитель транскрипции; действует на расстоянии в сотни–тысячи пар нуклеотидов от регулируемого гена. |
| **ЭПС (эндоплазматическая сеть)** | Система мембранных канальцев: шероховатая (с рибосомами — синтез белка) и гладкая (синтез липидов, стероидов). |
| **Эукариоты** | Организмы с оформленным ядром и мембранными органоидами; рибосомы 80S. |
| **Эухроматин** | Деспирализованный, транскрипционно активный хроматин (в центре ядра). |
| **Ядро** | Двумембранный органоид с порами; содержит нуклеоплазму, хроматин и ядрышко; хранит и реализует наследственную информацию. |
| **Ядрышко** | Структура ядра, где синтезируется рРНК и собираются субъединицы рибосом. |

## Раздел II. Генетика (94)

| Термин | Определение |
| --- | --- |
| ★ **Аллельные гены** | Гены, расположенные в одинаковых участках (локусах) гомологичных хромосом и контролирующие развитие вариаций одного признака. |
| ★ **Анализирующее скрещивание** | Скрещивание организма с доминантным признаком, но неизвестным генотипом, с организмом, имеющим рецессивный признак (для определения генотипа). |
| **Анеуплоидия** | Изменение числа хромосом по отдельным парам: трисомия (синдром Дауна 47,+21), моносомия (синдром Шерешевского–Тёрнера 45,X0). |
| **Аутбридинг** | Неродственное скрещивание; повышает гетерозиготность, даёт гетерозис, подавляет вредные рецессивы. |
| **Биохимический метод** | Выявление наследственных нарушений обмена по дефектным ферментам и метаболитам. Основа — гипотеза «один ген — один фермент» (Бидл, Татум). |
| **Близнецовый метод** | Сравнение моно- и дизиготных близнецов для разграничения роли наследственности и среды (Гальтон, 1876). |
| ★ **Гемизигота** | Организм, у которого в диплоидном наборе присутствует лишь один ген из аллельной пары (напр., гены Х-хромосомы у мужчин). |
| **Генеалогический метод** | Составление и анализ родословных для установления наследственной природы и типа наследования признака (Гальтон). |
| **Генеративные (гаметические) мутации** | Мутации в половых клетках, передающиеся потомству (гемофилия, брахидактилия). |
| **Генетика** | Наука о двух фундаментальных свойствах организмов — наследственности и изменчивости. |
| **Генетика человека** | Наука о роли наследственности и изменчивости в формировании нормальных признаков человека. |
| ★ **Генетическая карта хромосомы** | Отрезок прямой, на котором обозначены порядок взаимного расположения и расстояния между генами в хромосоме. |
| **Генетический груз** | Накопление в популяции вредных аллелей — «плата» за полиморфизм: мутационный, сегрегационный, субституционный. |
| **Генетический полиморфизм** | Присутствие в популяции двух и более аллелей (генотипов) с частотой более 1 %; адаптационный (переходный) и балансированный (устойчивый). |
| **Генные (точковые) мутации** | Изменения нуклеотидов в пределах гена (замена, вставка, выпадение) → новые аллели (серповидноклеточная анемия — замена 1 аминокислоты). |
| **Генокопии** | Сходный фенотип, обусловленный мутациями разных генов (гемофилия A и B, разные формы наследственной глухоты). |
| **Геномные мутации** | Изменения числа хромосом: полиплоидия, анеуплоидия, гаплоидия. Механизм — нерасхождение хромосом в мейозе/митозе. |
| ★ **Генотип** | Совокупность всех генов организма. |
| **Гетерогаметный пол** | Пол с разными половыми хромосомами (XY) или одной (X0); образует два типа гамет. |
| ★ **Гетерозиготный организм** | Организм, имеющий два разных аллельных гена и продуцирующий два типа гамет (Aa). |
| **Гетерозис** | Повышенная жизнеспособность и продуктивность гибридов первого поколения при аутбридинге. |
| **Гибридологический метод** | Метод изучения наследования путём скрещиваний и статистического анализа потомства (основной метод Менделя). |
| ★ **Голандрическое наследование** | Наследование признаков, гены которых локализованы в Y-хромосоме в гемизиготном состоянии (передаётся от отца всем сыновьям). |
| **Гомогаметный пол** | Пол с двумя одинаковыми половыми хромосомами (XX); образует один тип гамет. |
| ★ **Гомозиготный организм** | Организм, имеющий два одинаковых аллельных гена и продуцирующий один тип гамет (AA или aa). |
| **Группа сцепления** | Все гены одной хромосомы; число групп сцепления равно гаплоидному числу хромосом. |
| **Делеция** | Хромосомная мутация — утрата (выпадение) участка хромосомы. |
| ★ **Дигибридное скрещивание** | Скрещивание организмов с учётом двух пар альтернативных признаков. |
| **Дизиготные близнецы** | Близнецы из разных зигот; генетически как обычные сибсы, могут быть разнополыми. |
| **Дискордантность** | Различие близнецов по изучаемому признаку. |
| **Доминантный признак (аллель)** | Признак (аллель), подавляющий проявление альтернативного рецессивного у гетерозиготы; обычно соответствует синтезу белка. |
| **Дупликация** | Хромосомная мутация — удвоение участка хромосомы. |
| **Зависимое от пола наследование** | Признак проявляется у обоих полов, но по-разному (облысение: доминантно у мужчин, рецессивно у женщин). |
| **Закон Харди–Вайнберга** | В идеальной популяции частоты аллелей и генотипов постоянны: p+q=1 и p²+2pq+q²=1. Условия: большая численность, панмиксия, отсутствие мутаций, миграций, отбора. |
| **Изменчивость** | Свойство организмов изменять наследственные задатки или их проявление в онтогенезе. |
| **Инбридинг** | Близкородственное скрещивание; повышает гомозиготность, проявление вредных рецессивов, ведёт к вырождению. |
| **Инверсия** | Хромосомная мутация — поворот участка хромосомы на 180°. |
| **Канцерогенез** | Процесс зарождения и развития опухоли, часто через соматические мутации. |
| **Кодоминирование** | Оба аллеля проявляются независимо, формируя новый признак (IV группа крови AB — синтез обоих антигенов). |
| **Комбинативная изменчивость** | Наследственная изменчивость за счёт новых сочетаний генов при половом размножении. Механизмы: независимое расхождение хромосом, кроссинговер, случайное оплодотворение. |
| ★ **Комплементарность (генов)** | Тип взаимодействия неаллельных генов, при котором два неаллельных гена дополняют действие друг друга, формируя новый признак, отсутствующий у родителей. |
| **Конкордантность** | Сходство близнецов по изучаемому признаку (используется для оценки наследуемости). |
| **Локус** | Место (участок) расположения конкретного гена в хромосоме. |
| **Медико-генетическое консультирование (МГК)** | Предупреждение рождения больного ребёнка: уточнение диагноза → расчёт генетического риска → рекомендации. |
| **Медицинская генетика** | Наука о роли наследственности и изменчивости в патологии человека. |
| **Менделирующие признаки** | Признаки, наследующиеся по законам Менделя (альбинизм, цвет глаз, форма волос, группы крови, ряд болезней). |
| **Множественный аллелизм** | Существование у гена более двух аллелей (результат многократных мутаций локуса); пример — система AB0 (Iᴬ, Iᴮ, I⁰). |
| **Модификационная изменчивость** | Ненаследственное изменение признаков под действием среды, не затрагивающее генотип; групповая, направленная, обратимая (загар, гипертрофия мышц). |
| **Молекулярно-генетический метод** | Выделение и анализ ДНК/РНК (ПЦР, рестриктазы, секвенирование); точная ДНК-диагностика наследственной патологии. |
| **Моногибридное скрещивание** | Скрещивание организмов, различающихся по одной паре альтернативных признаков. |
| **Монозиготные близнецы** | Близнецы из одной зиготы; генетически идентичны, всегда одного пола. |
| **Морганида (сантиморган)** | Единица расстояния между генами на генетической карте, соответствующая 1 % кроссинговера. |
| **Мультифакториальные болезни** | Болезни с наследственной предрасположенностью, развивающиеся при взаимодействии многих генов и среды (гипертония, ИБС, язвенная болезнь, бронхиальная астма). |
| **Мутагенез** | Процесс возникновения мутаций; спонтанный (ошибки репликации) и индуцированный (под действием мутагенов). |
| **Мутагены** | Факторы, вызывающие мутации: физические (ионизирующее излучение, УФ), химические (иприт, аналоги оснований), биологические (вирусы, транспозоны). |
| **Мутация** | Качественное или количественное изменение ДНК (генотипа); внезапное, наследуемое, ненаправленное. Термин ввёл де Фриз (1901). |
| **Наследование** | Процесс передачи и реализации наследственной информации в поколениях. |
| **Наследственность** | Свойство организмов обеспечивать материальную преемственность признаков в ряду поколений. |
| **Наследственные болезни** | Группы: генные (моногенные), хромосомные, мультифакториальные; отдельно — митохондриальные (цитоплазматические). |
| ★ **Неаллельные гены** | Гены, расположенные в разных участках (локусах) хромосом и контролирующие развитие разных признаков. |
| **Неполное доминирование** | У гетерозиготы промежуточное проявление признака (ночная красавица: красный × белый → розовый). |
| **Норма реакции** | Генетически заданные пределы изменчивости признака при данном генотипе; наследуется не сам признак, а норма реакции. Узкая (группа крови) и широкая (масса тела). |
| **Ограниченное полом наследование** | Гены локализованы в аутосомах, но проявляются только у одного пола (молочность, форма таза). |
| **Пенетрантность** | Вероятность (частота) проявления гена у его носителей, выраженная в процентах. |
| **Плейотропия** | Влияние одного гена на несколько признаков (синдром Марфана: «паучьи пальцы» + дефект хрусталика + патология сердца). |
| ★ **Полигибридное скрещивание** | Скрещивание организмов с учётом нескольких (двух или более) пар альтернативных признаков. |
| ★ **Полимерия** | Тип взаимодействия неаллельных генов, при котором несколько неаллельных генов влияют на формирование одного (обычно количественного) признака. |
| **Полиплоидия** | Кратное увеличение гаплоидного набора хромосом (3n, 4n). |
| **Полное доминирование** | Доминантный аллель полностью подавляет рецессивный: гетерозигота Aa по фенотипу не отличается от AA. |
| **Популяционно-статистический метод** | Изучение частот генов и генотипов в популяциях; оценка распространённости (в т.ч. патологических) генов. |
| **Правило «чистоты гамет»** | В каждую гамету попадает только один из пары аллельных генов (т.к. гомологичные хромосомы расходятся в мейозе). |
| **Пренатальная диагностика** | Дородовое выявление патологии плода: амниоцентез, биопсия хориона, кордоцентез, УЗИ, фетоскопия. |
| **Пробанд** | Лицо, обратившееся за медико-генетической консультацией, с которого начинают составление родословной. |
| **Репарация** | Система исправления повреждений ДНК по неповреждённой комплементарной нити. Механизмы: фотореактивация, эксцизионная (тёмновая), пострепликативная. Дефект → пигментная ксеродерма. |
| **Рецессивный признак (аллель)** | Признак (аллель), не проявляющийся у гетерозиготы (подавляется доминантным); проявляется только в гомозиготе aa. |
| **Решётка Пеннета** | Таблица для определения возможных сочетаний гамет и генотипов потомства. |
| **Сверхдоминирование** | Гетерозигота превосходит обе гомозиготы (основа гетерозиса; серповидноклеточность: Aa устойчивы к малярии). |
| **Сибсы** | Родные братья и сёстры (потомки одних родителей). |
| **Соматические мутации** | Мутации в клетках тела, не передающиеся при половом размножении; важны для онкогенеза и вегетативного размножения. |
| **Сцепление генов** | Совместное наследование генов, расположенных в одной хромосоме. Полное (кроссинговера нет) и неполное. |
| **Сцепленное с полом наследование** | Наследование генов, локализованных в половых хромосомах. Х-сцепленные рецессивные (гемофилия, дальтонизм) чаще проявляются у мужчин. |
| ★ **Тератогенные факторы** | Факторы внешней среды, действующие во время беременности и приводящие к развитию врождённых пороков у зародыша (алкоголь, никотин, краснуха, радиация). |
| **Транслокация** | Хромосомная мутация — обмен участками между негомологичными хромосомами. |
| **Фенокопии** | Ненаследственные фенотипические изменения, имитирующие мутации (вызваны средой/тератогенами в критические периоды). |
| ★ **Фенотип** | Совокупность всех признаков организма, формирующихся в результате реализации генотипа в определённых условиях внешней среды. |
| **Хромосомная теория наследственности** | Теория Моргана: гены расположены в хромосоме линейно, каждый в своём локусе; гены одной хромосомы образуют группу сцепления; сцепление нарушается кроссинговером. |
| **Хромосомные мутации (аберрации)** | Изменения структуры хромосом: делеция, дупликация, инверсия, транслокация. |
| **Цитогенетический метод** | Изучение кариотипа под микроскопом; диагностика хромосомных болезней, определение пола. |
| **Цитоплазматическая наследственность** | Передача признаков через ДНК митохондрий/пластид (вне ядра); наследуется только по материнской линии (атрофия зрительного нерва Лебера). |
| **Экспрессивность** | Степень фенотипической выраженности признака у разных носителей гена. |
| ★ **Эпистаз** | Тип взаимодействия неаллельных генов, при котором ген одной пары (эпистатический, супрессор) подавляет действие неаллельного гена из другой пары (гипостатического). |
| **I закон Менделя (единообразия)** | При скрещивании гомозигот, различающихся по паре признаков, всё первое поколение (F₁) единообразно по фенотипу и генотипу. |
| **II закон Менделя (расщепления)** | При скрещивании гибридов F₁ в F₂ идёт расщепление 3:1 по фенотипу (1:2:1 по генотипу) при полном доминировании. |
| **III закон Менделя (независимого комбинирования)** | Гены разных аллельных пар, лежащие в разных хромосомах, наследуются независимо; F₂ дигибрида — 9:3:3:1. |

## Раздел III. Индивидуальное развитие (онтогенез) (47)

| Термин | Определение |
| --- | --- |
| **Аллантоис** | Зародышевый орган выделения и газообмена. |
| **Амнион** | Водная (защитная) оболочка зародыша. |
| **Апоптоз** | Запрограммированная гибель клеток — механизм морфогенеза в онтогенезе. |
| **Биологическая смерть** | Необратимый этап умирания с гибелью коры мозга и тканей. |
| **Биологические ритмы** | Закономерное повторение биологических процессов во времени; экзогенные и эндогенные («биологические часы»). |
| **Бластомеры** | Клетки, образующиеся при дроблении зиготы. |
| **Бластопор** | Первичный рот — отверстие гаструлы в первичную кишку. У первичноротых → рот, у вторичноротых → анус. |
| **Бластоцель** | Первичная полость бластулы. |
| **Бластула** | Однослойный зародыш с полостью (бластоцель); тип зависит от количества и распределения желтка. |
| **Врождённые пороки развития (ВПР)** | Структурные нарушения, возникшие пренатально и нарушающие функцию органа («заячья губа», «волчья пасть», пороки сердца). |
| **Гаструла** | Двух-/трёхслойный зародыш с полостью первичной кишки (гастроцель) и первичным ртом (бластопор). |
| **Гаструляция** | Образование гаструлы — двух-, затем трёхслойного зародыша (зародышевых листков). |
| **Голобластическое (полное) дробление** | Дробится всё яйцо (мало желтка); равномерное (ланцетник) и неравномерное (амфибии, млекопитающие). |
| ★ **Дробление** | Ряд последовательных митотических делений зиготы без роста бластомеров (объём не растёт, клетки мельчают); завершается образованием бластулы. |
| **Зародышевые листки** | Эктодерма, энтодерма, мезодерма — источники всех тканей и органов. |
| ★ **Клиническая смерть** | Обратимый этап умирания: прекращение дыхания, сердечной деятельности, угасание жизненно важных функций (обратима ~3–6 мин до гибели мозга). |
| ★ **Критические периоды развития** | Периоды эмбриогенеза с повышенной чувствительностью зародыша к повреждающему действию факторов внешней среды (совпадают с интенсивным органогенезом). |
| **Мезодерма** | Третий зародышевый листок. Способы закладки: телобластический (первичноротые) и энтероцельный (вторичноротые → целом). |
| **Меробластическое (неполное) дробление** | Дробится лишь часть яйца (много желтка); дискоидальное (птицы, рыбы) и поверхностное (насекомые). |
| **Морфаллаксис** | Восстановление органа путём перестройки оставшейся части (гидра, планария). |
| **Морфогенез** | Формирование структуры организма за счёт миграции, дифференцировки и избирательной гибели клеток. |
| **Нейрула** | Стадия зародыша с осевым комплексом зачатков: нервная трубка, хорда, первичная кишка, сомиты мезодермы. |
| **Нейруляция** | Образование нервной трубки из нервной пластинки эктодермы (нервная пластинка → желобок → трубка). |
| **Непрямое развитие (с метаморфозом)** | Развитие через личинку с провизорными органами и последующим превращением (лягушка: головастик; насекомые). |
| ★ **Онтогенез** | Совокупность взаимосвязанных и детерминированных хронологических событий индивидуального развития организма от зиготы (образования) до смерти. |
| **Плацента** | Орган обмена веществ между матерью и плодом (у человека — гемохориальная). |
| **Преформизм** | Устаревшая концепция: организм предобразован (заложен в готовом виде) в гамете. |
| **Провизорные органы** | Временные (зародышевые) органы: желточный мешок, амнион, хорион, аллантоис, плацента; после редуцируются. |
| **Производные мезодермы** | Скелет, мышцы, соединительная ткань (дерма), кровеносная, выделительная и половая системы. |
| **Производные эктодермы** | Нервная система, эпидермис и его производные (волосы, ногти, железы), эмаль зубов, органы чувств. |
| **Производные энтодермы** | Эпителий кишечника, печень, поджелудочная железа, эпителий лёгких и дыхательных путей. |
| **Прямое развитие** | Развитие, при котором потомок похож на взрослого и нуждается лишь в росте (млекопитающие, птицы). |
| **Реаниматология** | Наука об оживлении организма в терминальных состояниях; реанимация эффективна в период клинической смерти. |
| **Регенерация** | Восстановление утраченных или повреждённых структур; физиологическая и репаративная. |
| **Репаративная регенерация** | Восстановление структур после повреждения. Способы: эпиморфоз, морфаллаксис, регенерационная и компенсаторная гипертрофия. |
| **Рост** | Увеличение массы и размеров организма за счёт пролиферации и роста клеток. |
| **Способы гаструляции** | Инвагинация (впячивание), иммиграция (выселение клеток внутрь), деламинация (расщепление), эпиболия (обрастание). |
| ★ **Старение** | Универсальный процесс снижения уровня функционирования сложных, недостаточно самообновляющихся систем во времени, повышающий вероятность смерти. |
| **Типы онтогенеза** | Личиночный (с метаморфозом — насекомые, амфибии), неличиночный (яйцекладный — рыбы, рептилии, птицы), внутриутробный (плацентарные, человек). |
| **Физиологическая регенерация** | Естественное обновление функционирующих структур в норме (эритроциты, эпителий кожи и кишечника). |
| **Хорион** | Наружная оболочка зародыша (дыхание, питание); у млекопитающих участвует в образовании плаценты. |
| **Хронобиология** | Наука о биологических ритмах; прикладной раздел — хрономедицина (хронодиагностика, хронотерапия). |
| **Циркадные ритмы** | Околосуточные ритмы с периодом ~24 ч (сон/бодрствование, температура тела, секреция гормонов, митотическая активность). |
| **Эмбриогенез** | Период развития от оплодотворения (зиготы) до рождения: дробление → бластула → гаструла → гисто- и органогенез. |
| **Эмбриональная индукция** | Взаимодействие частей зародыша, при котором один зачаток направляет дифференцировку другого. |
| **Эпигенез** | Концепция: структуры организма формируются заново в ходе развития из недифференцированного материала. |
| **Эпиморфоз** | Отрастание органа от ампутационной поверхности через бластему (конечность тритона). |

## Раздел IV. Эволюционное учение (65)

| Термин | Определение |
| --- | --- |
| **Автономизация онтогенеза** | Снижение зависимости развития организма от факторов внешней среды. |
| **Адаптация** | Приспособление организмов к условиям среды: морфологические, физиологические, поведенческие; преадаптации и постадаптации. |
| **Аллопатрическое видообразование** | Видообразование через пространственную (географическую) изоляцию популяций. |
| **Анаболия (надставка)** | Эволюционное изменение на поздних стадиях развития; полно рекапитулирует филогенез. |
| **Аналогичные органы** | Органы разного происхождения, но сходной функции (крыло птицы и крыло бабочки; жабры рыбы и рака). |
| **Антропогенез** | Процесс эволюционного происхождения человека. Факторы: биологические (отбор) и социальные (труд, речь, мышление). |
| **Ароморфоз (арогенез)** | Крупное эволюционное приобретение, повышающее уровень организации (челюсти, лёгкие, теплокровность, кора мозга). |
| **Архаллаксис** | Эволюционное изменение на ранних стадиях эмбриогенеза (нарушает биогенетический закон). |
| **Архантропы** | Древнейшие люди (Homo erectus: питекантроп, синантроп); владели огнём, изготавливали орудия. |
| **Атавизмы** | Проявление у потомков признаков далёких предков (хвост, многососковость, сплошной волосяной покров). |
| **Бинарная номенклатура** | Двойное латинское название вида (родовое + видовое), введённое К. Линнеем. |
| **Биогенетический закон** | Мюллер–Геккель (1866): онтогенез есть краткое и быстрое повторение филогенеза. |
| **Биологический прогресс** | Процветание группы (Северцов): рост численности, расширение ареала, увеличение числа подчинённых групп. |
| **Биологический регресс** | Упадок группы: снижение численности, сужение ареала, вымирание. |
| **Борьба за существование** | Совокупность взаимоотношений организма со средой и другими организмами: межвидовая, внутривидовая, борьба с условиями среды. |
| **Вид** | Совокупность особей со сходными признаками, свободно скрещивающихся и дающих плодовитое потомство, занимающих общий ареал. |
| **Видообразование** | Процесс возникновения новых видов; аллопатрическое (географическое) и симпатрическое. |
| **Вторичноротые (Deuterostomia)** | Животные, у которых бластопор становится анусом, рот прорывается заново; мезодерма энтероцельно; нервная система спинная (иглокожие, хордовые). |
| **Генофонд** | Совокупность всех генов (генотипов) популяции или вида. |
| **Гомодинамия** | Гомология метамерных (повторяющихся вдоль оси тела) органов (позвонки, сегменты). |
| **Гомологичные органы** | Органы общего происхождения (из одного зачатка), но разной функции (рука человека, крыло птицы, ласт кита). |
| **Дарвинизм** | Теория эволюции Дарвина (1859): движущие силы — наследственность, изменчивость, борьба за существование, естественный отбор. |
| **Движущий отбор** | Отбор, сдвигающий среднее значение признака при направленном изменении среды (берёзовая пяденица). |
| **Дем** | Небольшая популяция человека (1500–4000 чел.) с высокой долей внутригрупповых браков. |
| **Дивергенция** | Расхождение признаков потомков от общего предка → разные таксоны; даёт гомологичные органы. |
| **Дизруптивный отбор** | Разрывающий отбор — против средних вариантов в пользу крайних. |
| **Дрейф генов** | Случайное изменение частот аллелей в малых популяциях. Эффекты: «бутылочное горлышко» и «эффект основателя». |
| **Естественный отбор** | Избирательное выживание и размножение наиболее приспособленных генотипов; единственный направляющий фактор эволюции. |
| **Закон зародышевого сходства** | К. Бэр (1828): эмбрионы животных одного типа на ранних стадиях сходны; общие признаки появляются раньше частных. |
| **Идиоадаптация (аллогенез)** | Частное приспособление к среде без повышения уровня организации (защитная окраска, форма клюва вьюрков). |
| **Изолят** | Малая популяция человека (менее 1500 чел.) практически без притока генов извне; высокая гомозиготность, дрейф генов. |
| **Изоляция** | Ограничение свободного скрещивания: географическая (пространственная) и биологическая (репродуктивная). Закрепляет различия популяций. |
| **Конвергенция** | Схождение признаков у неродственных групп в сходных условиях; даёт аналогичные органы. |
| **Критерии вида** | Морфологический, физиологический, биохимический, генетический, экологический, географический, этологический. |
| **Ламаркизм** | Первая целостная теория эволюции (Ламарк): движущие силы — упражнение/неупражнение органов и наследование приобретённых признаков (ошибочно). |
| **Макроэволюция** | Образование надвидовых таксонов — родов, семейств и более крупных групп. |
| **Микроэволюция** | Эволюционные изменения внутри вида (на уровне популяций), ведущие к видообразованию. |
| **Мутационный процесс** | Поставщик элементарного эволюционного материала — новых аллелей и генов. |
| **Неоантропы** | Современные люди (кроманьонцы, Homo sapiens sapiens); членораздельная речь, искусство, жилища. |
| **Общая дегенерация (катагенез)** | Упрощение организации (часто при паразитизме или сидячем образе жизни — редукция органов у паразитических червей). |
| **Палеоантропы** | Древние люди (неандертальцы); развитая орудийная деятельность, охота, зачатки речи. |
| **Палингенезы** | Признаки зародыша, рекапитулирующие филогенез (жаберные щели, хорда у зародыша). |
| **Параллелизм** | Независимое развитие сходных признаков у родственных групп (ласты у ихтиозавров и китообразных). |
| **Первичноротые (Protostomia)** | Животные, у которых бластопор становится ртом; мезодерма телобластически; нервная система брюшная (черви, моллюски, членистоногие). |
| **Популяционные волны** | «Волны жизни» — периодические колебания численности популяции (Четвериков), случайно меняющие частоты аллелей. |
| **Популяция** | Совокупность особей одного вида, длительно живущих на территории, свободно скрещивающихся (панмиксия) и дающих плодовитое потомство. |
| **Поток генов (миграции)** | Перемещение особей/гамет между популяциями, выравнивающее различия их генофондов. |
| **Раса** | Совокупность людей с общностью наследуемых физических признаков, связанных с определённым ареалом. Три большие расы: европеоидная, негроидная, монголоидная. |
| **Расизм** | Антинаучное учение о неравноценности человеческих рас. Расы биологически равноценны, различия адаптивны и поверхностны, все люди — единый вид. |
| **Рекапитуляция** | Повторение признаков предков в онтогенезе потомков. |
| **Рудименты** | Недоразвитые органы, утратившие основную функцию (копчик, аппендикс, мышцы ушной раковины). |
| **Сбалансированный полиморфизм** | Длительное сохранение в популяции нескольких аллелей за счёт преимущества гетерозигот (серповидноклеточность в малярийных районах). |
| **Симпатрическое видообразование** | Видообразование в пределах одного ареала (экологическая/генетическая изоляция, полиплоидия). |
| **Синтетическая теория эволюции (СТЭ)** | Современный синтез дарвинизма и популяционной генетики (1920–1950). |
| **Стабилизирующий отбор** | Отбор против крайних вариантов признака при постоянстве условий среды. |
| **Субституция органов** | Замещение одного органа другим в филогенезе с сохранением функции (хорда → позвоночник; первичная почка → вторичная). |
| **Таксоны** | Соподчинённые систематические категории: вид → род → семейство → отряд → класс → тип → царство (у растений — порядок и отдел). |
| **Филогенетические корреляции** | Сопряжённые (координированные) изменения органов в филогенезе: топографические, динамические, биологические. |
| **Филэмбриогенезы** | Учение Северцова: эволюция через преобразования эмбриогенеза — архаллаксис, девиация, анаболия. |
| **Формы эволюции групп** | Основа макроэволюции: дивергенция, конвергенция, параллелизм, филетическая эволюция. |
| **Ценогенезы** | Приспособления зародыша/личинки к среде, не сохраняющиеся у взрослого (амнион, плацента, роговые зубы головастика). |
| **Эволюционная морфология** | Наука о закономерностях филогенетических преобразований органов. Метод тройного параллелизма (сравнительная анатомия + эмбриология + палеонтология). |
| **Эволюционное учение** | Система представлений о причинах и закономерностях исторического развития живой природы. |
| **Элементарные эволюционные факторы** | Мутационный процесс, миграции (поток генов), дрейф генов, популяционные волны, изоляция, естественный отбор. |
| ★ **Эмбрионизация онтогенеза** | Возникновение в ходе эволюции способности проходить значительную часть зародышевого развития под защитой материнского организма или зародышевых оболочек. |

## Раздел V. Филогенез систем органов хордовых (16)

| Термин | Определение |
| --- | --- |
| **Вольфов канал** | Проток первичной почки; у самцов млекопитающих становится семяизвергательным каналом, у самок редуцируется. |
| **Гетеродонтная зубная система** | Зубы, дифференцированные на резцы, клыки и коренные (млекопитающие). |
| **Гомодонтная зубная система** | Однотипные недифференцированные зубы (рыбы, амфибии, рептилии); у человека — атавизм. |
| **Дивертикул Меккеля** | Порок: вырост подвздошной кишки — остаток желточного протока. |
| **Жаберные артериальные дуги** | Закладывающиеся 6 пар сосудов у зародышей позвоночных. У млекопитающих: 3-я → сонные артерии, 4-я левая → дуга аорты, 6-я → лёгочные артерии. |
| **Зауропсидный тип мозга** | Тип мозга рептилий и птиц; ведущий отдел — полосатые тела переднего мозга (появляется древняя кора, archicortex). |
| **Ихтиопсидный тип мозга** | Тип головного мозга рыб и амфибий; ведущий отдел — средний мозг. |
| **Крипторхизм** | Порок мочеполовой системы: неопущение яичка в мошонку. |
| **Маммалийный тип мозга** | Тип мозга млекопитающих; ведущий отдел — новая кора (neocortex), есть межполушарная комиссура. |
| **Мезонефрос (первичная почка)** | Туловищная почка взрослых рыб и амфибий; появляются капсулы клубочков. |
| **Метанефрос (вторичная почка)** | Тазовая почка рептилий и млекопитающих; у млекопитающих появляется петля Генле. |
| **Мюллеров канал** | Проток, дающий у самок яйцевод, матку и влагалище; у самцов редуцируется. Неполное слияние → двурогая/двойная матка. |
| **Пронефрос (предпочка)** | Головная почка; у личинок рыб и амфибий; нефроны открываются в целом. |
| **Сомиты** | Сегменты дорсальной мезодермы зародыша: дают склеротом (скелет), миотом (мышцы), дерматом (дерму). |
| **Хорда** | Осевой скелет низших хордовых и зародышей позвоночных; у человека даёт студенистое ядро межпозвонкового диска. |
| **Целом** | Вторичная полость тела; у вторичноротых (хордовых) закладывается энтероцельным способом. |

## Раздел VI. Биосфера и экология (26)

| Термин | Определение |
| --- | --- |
| **Агроценоз** | Искусственная неустойчивая экосистема (поле, сад) с малым разнообразием и разомкнутым круговоротом; зависит от энергии человека. |
| **Адаптивный тип (человека)** | Комплекс морфофизиологических признаков для приспособления к среде (Алексеева): арктический, тропический, горный, умеренного пояса. |
| **Биогенное вещество** | Продукты жизнедеятельности организмов: уголь, нефть, известняк. |
| **Биогеоценоз (экосистема)** | Устойчивая система из биоценоза и биотопа с круговоротом веществ; элементарная структурная единица биосферы. |
| **Биокосное вещество** | Вещество, созданное совместно организмами и неживыми процессами (почва). |
| **Биосфера** | Оболочка Земли, населённая живыми организмами и преобразованная ими (термин — Э. Зюсс, 1875; учение — В.И. Вернадский). Элементарная единица — биогеоценоз. |
| **Биотоп** | Участок среды с относительно однородными условиями обитания. |
| **Биоценоз** | Сообщество организмов, населяющих биотоп (фитоценоз, зооценоз, микробоценоз). |
| **Живое вещество** | Совокупность всех живых организмов биосферы; главный геохимический агент (Вернадский). |
| **Квартирантство** | Использование одного вида другим как местообитания без вреда (орхидеи на деревьях; рыбки в полости голотурий). |
| **Климакс** | Устойчивое конечное сообщество, которым завершается сукцессия. |
| **Комменсализм (сотрапезничество)** | Сожительство с пользой для одного вида и безразличием для другого («+0»): ротовая амёба; рыба-прилипала и акула. |
| **Консументы** | Гетеротрофы, потребляющие готовое органическое вещество (животные); I, II, III порядка. |
| **Косное вещество** | Вещество, образованное без участия живых организмов (горные породы). |
| **Мутуализм** | Взаимовыгодное сожительство видов («++»): бактерии кишечника и человек; клубеньковые бактерии и бобовые. |
| **Ноосфера** | «Сфера разума» (Вернадский) — этап эволюции биосферы, когда разумная деятельность человека становится определяющим фактором развития. |
| ★ **Паразитизм** | Сожительство, при котором паразит использует хозяина как местообитание и источник пищи, причиняя ему вред. |
| **Продуценты** | Автотрофы, создающие органическое вещество из неорганического (растения). |
| **Редуценты** | Организмы, разлагающие органические остатки до неорганических веществ (бактерии, грибы). |
| ★ **Симбиоз** | Одна из форм совместного существования двух организмов, принадлежащих к разным видам. |
| **Сукцессия** | Последовательная смена биоценозов на территории; первичная (на безжизненном субстрате) и вторичная (после нарушения); завершается климаксом. |
| **Функции биосферы** | Газовая (миграция O₂/CO₂), концентрационная (накопление элементов), окислительно-восстановительная, биохимическая. |
| **Хищничество** | Пищевые отношения, при которых хищник убивает и поедает жертву («+−»). |
| **Экологическая пирамида** | Графическое отображение трофических уровней. Правило 10 %: на каждый уровень переходит ~10 % энергии. Пирамиды энергии, биомассы, чисел. |
| **Экологический фактор** | Любой элемент среды, влияющий на организм: абиотические, биотические, антропогенные. |
| **Экология** | Наука о взаимоотношениях организмов между собой и со средой обитания. |

## Раздел VII. Медицинская паразитология (98)

| Термин | Определение |
| --- | --- |
| **Адолескарий** | Инцистированная инвазионная личинка фасциолы на траве или в воде. |
| **Альвеококкоз** | *Alveococcus multilocularis*; многокамерный узел, растёт инфильтративно как опухоль; окончательный хозяин — лисы, песцы. |
| **Амёбиаз** | Кишечная инфекция, возбудитель — дизентерийная амёба *Entamoeba histolytica*; кровавый понос, язвы кишки, абсцессы печени. Инвазионная форма — циста. |
| **Анкилостомоз** | *Ancylostoma duodenale*; перкутанное заражение филяриевидной личинкой; кровопотеря → железодефицитная анемия. |
| **Антропозооноз** | Болезнь, общая для человека и животных. |
| **Антропоноз** | Болезнь, источником возбудителя которой служит только человек. |
| **Аскаридоз** | Аскарида *Ascaris lumbricoides* (геогельминт); инвазионная стадия — яйцо с личинкой; миграция личинки через лёгкие. |
| **Аутоинвазия (аутореинвазия)** | Самозаражение хозяина яйцами или личинками паразита (свиной и карликовый цепни, острица). |
| **Балантидиаз** | Язвенный колит, возбудитель — инфузория *Balantidium coli*; резервуар — свиньи. |
| **Биогельминт** | Гельминт, развивающийся со сменой хозяев — с участием промежуточного хозяина (печёночный сосальщик, цепни). |
| **Ботрии** | Присасывательные щели на сколексе лентеца широкого (вместо присосок). |
| ★ **Возбудитель** | Биологический объект, который при проникновении в организм животного или человека вызывает заболевание. |
| **Временные паразиты** | Паразиты, контактирующие с хозяином лишь для питания (комары, клопы). |
| **Вухерериоз** | Филярия *Wuchereria bancrofti*; переносчик — комары; закупорка лимфатических сосудов → «слоновость» (элефантиаз). |
| **Геогельминт** | Гельминт, развивающийся без промежуточного хозяина: яйца/личинки дозревают во внешней среде, чаще в почве (аскарида, власоглав). |
| **Гименолепидоз** | Карликовый цепень *Hymenolepis nana*; инвазионная стадия — яйцо; характерна аутоинвазия; чаще у детей. |
| **Гинекофорный канал** | Желоб на теле самца шистосомы, в котором постоянно лежит шнуровидная самка. |
| **Гипостом** | Ротовой вырост иксодовых клещей с зубцами для прочной фиксации на хозяине. |
| **Гонотрофический цикл** | У кровососущих комаров: цикл «нападение и кровососание → переваривание крови и созревание яиц → откладка». |
| **Девастация** | Учение Скрябина — комплекс мероприятий по полному истреблению возбудителей гельминтозов как вида (в отличие от дегельминтизации — лечения особей). |
| **Дикроцелиоз** | Ланцетовидный сосальщик *Dicrocoelium lanceatum*; метацеркарий в муравье. ⚠ «Транзиторные» яйца при поедании печени не означают заражения. |
| **Дифиллоботриоз** | Лентец широкий *Diphyllobothrium latum*; плероцеркоид в рыбе; поглощает витамин B12 → B12-дефицитная анемия. |
| ★ **Донор** | Группа животных или человек, являющиеся источником распространения возбудителей заболевания. |
| **Дракункулёз** | Ришта *Dracunculus medinensis* (крупнейшая нематода); промежуточный хозяин — рачок-циклоп (питьё сырой воды); подкожные нарывы. |
| **Зооноз** | Болезнь, источником возбудителя которой служат животные. |
| **Имаго** | Взрослая (половозрелая) стадия насекомого или клеща. |
| **Инвазионная стадия** | Стадия развития паразита, способная заразить хозяина (напр., циста, метацеркарий, цистицерк, филяриевидная личинка). |
| **Инокуляция** | Попадание возбудителя в организм со слюной переносчика при укусе. |
| **Клещевой возвратный тиф** | Спирохетоз; переносчик — аргазовый (поселковый) клещ *Ornithodorus papillipes*; трансовариальная передача. |
| **Клещевой энцефалит** | Вирусная природно-очаговая болезнь; переносчик — иксодовый (таёжный) клещ; трансовариальная передача, передача через молоко коз. |
| **Контаминация** | Попадание возбудителя в организм с фекалиями переносчика через ранку или расчёс. |
| **Лейшманиоз** | Трансмиссивная болезнь (переносчик — москит *Phlebotomus*): кожный (*L. tropica*) и висцеральный (*L. donovani*). |
| **Лямблиоз** | Возбудитель *Lamblia intestinalis*; двенадцатиперстная кишка; нарушение пристеночного пищеварения, чаще у детей. |
| **Малярия** | Возбудитель *Plasmodium* spp.; окончательный хозяин — комар *Anopheles*; приступы лихорадки (vivax/ovale — 48 ч, malariae — 72 ч, falciparum — тяжёлая). |
| **Марита** | Половозрелая форма сосальщика (трематоды). |
| **Медицинская паразитология** | Наука о паразитах человека, вызываемых ими болезнях и мерах борьбы. Разделы: протозоология, гельминтология, арахноэнтомология. |
| **Метаморфоз** | Превращение в развитии насекомых: полный (яйцо → личинка → куколка → имаго) и неполный (без стадии куколки). |
| **Метацеркарий** | Инцистированная инвазионная личинка сосальщиков (в рыбе — описторх, в раках — парагонимус, в муравье — дикроцелий). |
| **Механический переносчик** | Членистоногое, переносящее возбудителя на покровах или в кишечнике без его развития (мухи, тараканы). |
| **Миаз** | Заболевание, вызванное развитием личинок мух в тканях и органах человека (вольфартова муха). |
| **Микрофилярии** | Личинки филярий, циркулирующие в крови (у Wuchereria — ночью, что связано с активностью комаров). |
| **Мирацидий** | Ресничная личинка сосальщика, выходящая из яйца и внедряющаяся в моллюска. |
| **Морфофизиологические адаптации к паразитизму** | Органы фиксации (присоски, крючья), редукция органов, мощная половая система, сложные циклы со сменой хозяев, антиферментная защита. |
| ★ **Окончательный (дефинитивный) хозяин** | Организм, в котором паразит находится в половозрелой форме или размножается половым путём. |
| **Онкосфера** | Шестикрючная зародышевая личинка ленточных червей, выходящая из яйца. |
| **Оотека** | Защитный кокон с яйцами у тараканов. |
| **Описторхоз** | Кошачий сосальщик *Opisthorchis felineus*; метацеркарий в карповых рыбах; поражение печени, жёлчного пузыря. |
| **Парагонимоз** | Лёгочный сосальщик *Paragonimus westermani*; метацеркарий в раках и крабах; поражение лёгких. |
| ★ **Паразит** | Организм, который живёт постоянно или временно за счёт других организмов и использует их как место обитания и источник питания. |
| ★ **Паразитология** | Наука, изучающая сложный комплекс явлений паразитизма. |
| **Педикулёз** | Вшивость (головная и платяная вши, *Pediculus humanus*). Платяная вошь — переносчик сыпного и возвратного тифа. |
| ★ **Переносчик** | Биологический объект, переносящий возбудителей заболевания от донора к реципиенту. |
| **Плероцеркоид** | Инвазионная личиночная стадия лентеца широкого в теле рыбы. |
| **Постоянные (стационарные) паразиты** | Паразиты, проводящие на хозяине всю жизнь или её большую часть (вши, гельминты). |
| ★ **Природно-очаговая болезнь** | Болезнь, возбудители которой неопределённо долго циркулируют в дикой природе независимо от деятельности человека (резервуар — дикие животные). |
| ★ **Природный очаг заболевания** | Участок территории определённого ландшафта, на котором эволюционно сложились межвидовые взаимоотношения между возбудителем, донором, переносчиком и резервуаром. |
| **Проглоттиды (членики)** | Сегменты тела ленточного червя; зрелые проглоттиды содержат матку, заполненную яйцами. |
| ★ **Промежуточный хозяин** | Организм, в котором паразит находится в личиночной стадии или размножается бесполым путём. |
| **Процеркоид** | Первая личиночная стадия лентеца широкого в рачке-циклопе. |
| **Редия** | Личиночная стадия сосальщика в моллюске (партеногенетическое размножение). |
| ★ **Резервуар природного очага** | Совокупность донора, возбудителя, переносчика и реципиента при соответствующих экологических условиях. |
| ★ **Резервуарный хозяин** | Организм, в котором происходит накопление паразита без его дальнейшего развития (рыбы для лентеца широкого). |
| ★ **Реципиент** | Организм, заражаемый возбудителем заболевания. |
| **Сколекс** | Головка ленточного червя с органами фиксации (присоски, крючья или присасывательные щели — ботрии). |
| **Специфический переносчик** | Переносчик, в организме которого возбудитель проходит развитие и/или размножение (малярийный комар, москит). |
| **Спорогония** | Половое размножение споровиков с образованием спорозоитов (у малярийного плазмодия — в комаре Anopheles). |
| **Спороциста** | Личиночная стадия сосальщика в теле моллюска (бесполое размножение). |
| **Стробила** | Всё тело ленточного червя — цепочка члеников (проглоттид). |
| **Стронгилоидоз** | Угрица кишечная *Strongyloides stercoralis*; инвазионная — филяриевидная личинка; опасен при иммунодефиците. Диагноз — личинки в кале. |
| **Тегумент** | Наружный погружённый эпителий плоских паразитических червей; обеспечивает всасывание питательных веществ и антиферментную защиту. |
| **Тениаринхоз** | Бычий цепень *Taeniarhynchus saginatus* («невооружённый»); цистицерк в говядине. ⚠ Цистицеркоз у человека невозможен. |
| **Тениоз** | Свиной цепень *Taenia solium* («вооружённый»); цистицерк в свинине. |
| **Токсоплазмоз** | Возбудитель *Toxoplasma gondii*; окончательный хозяин — кошачьи; опасно трансплацентарное заражение (пороки мозга плода). |
| **Трансмиссивные болезни** | Болезни, передающиеся через переносчика — кровососущих членистоногих (малярия, лейшманиоз, клещевой энцефалит). |
| **Трансовариальная передача** | Передача возбудителя у членистоногих через яйца следующему поколению (иксодовые и аргазовые клещи). |
| **Трипаносомоз** | Африканский (сонная болезнь, *T. brucei*, переносчик — муха цеце) и американский (болезнь Шагаса, *T. cruzi*, триатомовые клопы). |
| **Трихинеллёз** | *Trichinella spiralis*; личинки инкапсулируются в мышцах (сырое мясо кабана, свиньи); диагноз — биопсия мышц. |
| **Трихомоноз** | Урогенитальная инфекция, возбудитель *Trichomonas vaginalis*; половой путь; цист не образует. |
| **Трихоцефалёз** | Власоглав *Trichocephalus trichiurus* (геогельминт); передний волосовидный конец внедряется в стенку слепой кишки. |
| **Трофозоит (вегетативная форма)** | Активная питающаяся и подвижная форма простейшего. |
| **Учение о природной очаговости** | Е.Н. Павловский (1939): ряд болезней имеет природные очаги, существующие независимо от человека; человек заражается, попадая в очаг. |
| **Фасциолёз** | Печёночный сосальщик *Fasciola hepatica*; инвазионная стадия — адолескарий; поражение жёлчных ходов, цирроз. |
| **Финна (ларвоциста)** | Пузырчатая личиночная стадия ленточных червей. Виды: цистицерк, цистицеркоид, эхинококковый пузырь, плероцеркоид. |
| **Фтириаз** | Паразитирование лобковой вши (площицы, *Phthirus pubis*) на участках с волосами. |
| **Хозяин** | Организм, за счёт которого живёт паразит, предоставляя ему место обитания и пищу. |
| **Церкарий** | Хвостатая личинка сосальщика, покидающая моллюска. У шистосом активно внедряется через кожу. |
| **Циста** | Покоящаяся стадия простейших, защищённая плотной оболочкой; инвазионная форма многих кишечных простейших (амёба, лямблия, балантидий). |
| **Цистицерк** | Финна в виде пузырька с одной ввёрнутой головкой (бычий и свиной цепни). |
| **Цистицеркоз** | Финны свиного цепня в тканях человека (мозг, глаза, мышцы) при заглатывании яиц (грязные руки, аутоинвазия при рвоте). Опасно. |
| **Чесотка** | Возбудитель — чесоточный зудень *Sarcoptes scabiei*; чесоточные ходы в эпидермисе; ночной зуд; прямой контактный путь. |
| **Чума** | Особо опасная инфекция; переносчик — блоха («чумной блок» в преджелудке), резервуар — грызуны. |
| **Шизогония** | Множественное бесполое деление (у споровиков, напр. эритроцитарная шизогония малярийного плазмодия). |
| **Шистосомоз** | Кровяные сосальщики *Schistosoma* spp. (раздельнополы); церкарий внедряется через кожу в воде; мочеполовой и кишечный. |
| **Эктопаразиты** | Паразиты, обитающие на поверхности тела хозяина (вши, блохи, чесоточный зудень). |
| ★ **Элементарный природный очаг** | Наименьший участок территории, на котором обеспечивается непрерывная циркуляция возбудителей природно-очаговых заболеваний. |
| **Эндопаразиты** | Паразиты, обитающие внутри хозяина: полостные, тканевые, внутриклеточные (малярийный плазмодий, лейшмании). |
| **Энтеробиоз** | Острица *Enterobius vermicularis*; перианальный зуд; диагноз — соскоб с перианальных складок (не кал); аутореинвазия. |
| **Эхинококкоз** | *Echinococcus granulosus*; человек — промежуточный хозяин; однокамерный пузырь (финна) в печени, лёгких. Заражение — яйцом от собак. |
